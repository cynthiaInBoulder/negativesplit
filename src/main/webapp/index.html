Negative Split Calculator for Garmin or GPX files

Instructions

  • Login into your connect.garmin.com account
  • Select file you wish to analyze
  • Select "Export" and "tcx" or "gpx" format
  • Save to your PC
  • Click on "Choose File" button to select file
  • Click on "Calculate"
